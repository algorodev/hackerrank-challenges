# Time Conversion

Given a time in 12-hour AM/PM format, convert it to military (24-hour) time.

### Example

```
12:00:00AM
```

### Return

```
00:00:00
```

### Function Description

timeConversion has the following parameter:

```
a time in  12-hour format
```

### Returns

```
the time in 24-hour format
```
